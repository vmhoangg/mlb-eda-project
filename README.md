# MLB Exploratory Data Analysis (EDA)

## Project Overview
This project performs an exploratory data analysis on Major League Baseball player statistics.

The analysis includes:
- Descriptive statistics
- Missing data handling
- Data visualization
- Chi-square test
- T-test
- Linear regression model

## Dataset
- 352 observations
- 21 variables
- Includes salary, performance metrics, demographic variables

## Key Findings
- Strong linear relationship between career at-bats and career hits
- Significant association between player position and race
- Regression model explains ~74% of variance in career triples

## Technologies Used
- Python
- Pandas
- Seaborn
- Statsmodels
- Scipy
